## Frontend

This repo contains the frontend(vue.js) for ilytics.sg 

## Run Instruction

- `npm install`
 
- `npm run serve`

## Auto-Deployment

This repo has CI/CD setup. 

To deploy manually:

- `npm run build` to generate `dist` folder

- Upload the content in `dist` to s3 ilytics bucket
